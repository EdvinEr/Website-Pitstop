Project Description

Pitstop

As of now, there is available data on the official F1 website, presenting historical data. However, this data is all presented in tables, which may make it overwhelming for the user to understand. Further, as the data is presented in tables it is likely to not be a pleasant experience for the user to interact with. Notably, there exists “Fantasy F1” which several users across the world play in, trying to predict possible winners in different race tracks. Having a web application which allows the users to easily compare and ultimately draw conclusions, can help the user in this regard.

On Formula 1’s website there is a lot of detailed data available. However, this data is only placed in various tables that can be filtered. There is no visualization of the data and no possibility to compare data over time between different seasons. The current available data only allows users to look at a specific year from 1950 to 2024, rather than making comparisons between them. The main functionality consists of filtering to look at points and positions for constructors (teams) and drivers in total and for specific races. When looking at a specific race, data about fastest lap times for practice, qualifying and the race can be examined.
